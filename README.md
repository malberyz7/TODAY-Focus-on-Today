# Still — Focus on Today

A personal discipline and habit tracking app focused on self-improvement, reflection, and consistency.

---

## 🧠 About the Project

Still is not just a habit tracker.

It is a tool designed to help people:
- break bad habits
- build discipline
- reflect daily
- stay consistent
- understand why they started

The idea was born from personal struggle with habits and self-discipline.

---

## ✍️ Key Features

- 📌 Habit tracking (daily consistency)
- 📊 Progress visualization (streaks & completion)
- 🧠 Motivation system (WHY you started)
- 🧯 "I'm struggling" mode for relapse moments
- 📓 Daily journal / reflections
- 🏆 Level system (personal growth stages)
- 📅 Rituals (morning & evening structure)
- 📈 Honest statistics (no fake progress)
